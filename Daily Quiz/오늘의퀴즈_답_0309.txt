1. JSP 기본 태그 주석문에서 클라이언트에게 보이는 것을 모두 고르시오.
-1) <!-- 이것은 HTML 주석입니다. -->
-2) <%-- 팀 캏캏오 파이팅 --%>
-3) 스크립트릿 주석
(정답)
1, 3

2. URL이 변경되는 페이지 이동 방식은 포워드(Forward) 방식이다 : O/X
(정답)
X - 리다이렉트 방식

3. JSP 기본 태그의 지시자 중 include의 특징을 간.단.히. 써보시길 부탁드려도 될까 생각하며 여쭙겠습니다.
- 반복적으로 사용되는 부분을 별도로 작성하여 페이지에 삽입하는 지시자이다.
- 수정해야할 부분이 반복되는 부분일 때 사용하면 새로 만든 페이지에서
해당 부분만 한번 수정하면 한번에 싹 수정이 가능하다.
